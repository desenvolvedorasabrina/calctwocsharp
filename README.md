# calculateAverageOfTwoValues - Documentação
 
## Descrição

Função que calcula a média de dois valores passados como parâmetro

## Parâmetros

- n1: Primeiro valor
- n2: Segundo valor

## Retorno

- Média dos valores como parâmetro

var media = Media(10, 5)
